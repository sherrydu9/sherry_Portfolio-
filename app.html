OVERVIEW

This is a track app that I design for the puppy lovers. Users can drop the pin on the map to record the puppies that they meet. Also users are able to create their customized list of dogs, and make further edits. This is a mobile-first application which optimizes for the mobile experience.

BACKGROUND

An animal tracking application to track different breed puppies in the city. This application study cover how to build Web apps that will scale to different screens using CSS3 and emerging CSS standards such as fluid design.

PROBLEMS

1. Dogs like to play and make friends with other dogs. How do they discover the whereabouts of other dogs?

2. Dog adoption is the best way for people who love dog and also want to save dog, but how does dog lover find them?

3. Without the ability to find the same breed, how do dogs find their lover?

SOLUTIONS

A social application that helps dog owners connect with other dogs locally and in other areas.

FINAL DESIGN

Let the fun begin

Sign in to puppy tracker app and join the tracking community to discover more and more puppies and places.

Powerful search bar and filters

Users are able to search and filter through all the puppies to quickly find the dog that they want. Filters include gender and size.

View puppy Profiles

Tap on any puppy icon on the map (or on the list view) and see all the details about them, including where they appeared on the map previously.

Add a new puppy

If you spot a puppy in the city, go ahead and pin them on the map and add their details. After adding, user can check the list anytime.

Edit an existing profile

Puppies are always on the run and changing probably has different breeds requests for them all the time. Users are able to edit existing data and keep them most up-to-date.
