Motion Graphic of Hello Body
Motion Graphic / Visual

OVERVIEW

The project shows the basic principles of motion graphics using Adobe AfterEffects, which including 3D effects.

BACKGROUND

After wearing the special contact lenses, the technology analyzes the wearer’s body. The technology will then be able to show the wearer’s body on a 3-dimensional model. This is a futuristic GUI composited onto live footage.

VIDEO

This motion graphic video that I made which is called Hello Body in 2019 for social media in Youtube.

CONCLUSION

For this project I taught myself how to make graphics with 3D effects in Adobe AfterEffects. I was always fascinated by 3D animations but didn't know how to do it myself. So when the assignment presented itself, I set a goal to learn the software and eventually did in a few weeks, thus was able to create this video.
